# 🎯 Google Cloud TTS 注册 - 步骤 3

## 当前状态：
✅ 已登录 Google Cloud Console
✅ 已启用 Text-to-Speech API

## 下一步：创建 API Key

### 操作步骤：

1. **进入 "凭据" 页面**
   - 左侧菜单："API和服务" → "凭据" / "Credentials"
   
   **或直接访问：**
   👉 https://console.cloud.google.com/apis/credentials

2. **点击 "创建凭据"**
   - 页面顶部有蓝色按钮 "+ 创建凭据" / "+ CREATE CREDENTIALS"
   - 点击后选择 "API 密钥" / "API key"

3. **复制 API Key**
   - 会弹出一个框，显示你的 API Key
   - 格式类似：`AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
   - **点击复制按钮**，保存好！

---

## ⚠️ 重要安全设置（推荐做）

### 限制 API Key（防止被盗用）：

1. **点击 "限制密钥"** / "Restrict Key"
2. **应用程序限制：**
   - 选择 "HTTP 引荐来源网址" / "HTTP referrers"
   - 添加：`localhost`
   - 添加：`localhost:8888`
3. **API 限制：**
   - 选择 "限制密钥" / "Restrict key"
   - 勾选 "Cloud Text-to-Speech API"
4. **点击 "保存"**
