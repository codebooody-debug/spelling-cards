## 📝 注册步骤

### 步骤 1: 创建 Google Cloud 账号
1. 访问: https://cloud.google.com
2. 点击 **"免费开始使用"**
3. 用 Google/Gmail 账号登录

### 步骤 2: 启用结算（需要信用卡）
⚠️ **需要信用卡验证**，但不会扣费
- 输入信用卡信息
- 获得 **$300 免费试用金**（90天有效）
- Text-to-Speech 有独立免费层，不计入试用金

### 步骤 3: 启用 Text-to-Speech API
1. 进入 Google Cloud Console
2. 点击左上菜单 → **"API和服务"** → **"库"**
3. 搜索 **"Text-to-Speech"**
4. 点击 **"启用"**

### 步骤 4: 创建 API Key
1. 点击菜单 → **"API和服务"** → **"凭据"**
2. 点击 **"创建凭据"** → **"API密钥"**
3. 复制生成的密钥

### 步骤 5: 限制 Key（安全）
1. 点击刚创建的 Key
2. **"应用程序限制"** → 选择 **"HTTP引荐来源网址"**
3. 添加你的域名（本地开发用 `localhost`）
4. **"API 限制"** → 选择 **"Cloud Text-to-Speech API"**
