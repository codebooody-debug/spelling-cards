# 🎯 Google Cloud TTS 注册 - 步骤 2

## 当前状态：
✅ 已登录 Google Cloud Console

## 下一步：启用 Text-to-Speech API

### 操作步骤：

1. **找到左侧导航菜单**
   - 点击左上角 ☰ 三条横线图标（如果菜单是收起状态）

2. **找到 "API和服务"**
   - 菜单里找 "APIs & Services" 或 "API和服务"
   - 鼠标悬停或点击展开

3. **点击 "库" (Library)**
   - 在展开的子菜单中点击 "库" / "Library"

4. **搜索 "Text-to-Speech"**
   - 在搜索框输入 "text to speech"
   - 找到 "Cloud Text-to-Speech API"

5. **点击启用**
   - 进入详情页后，点击蓝色 "启用" / "Enable" 按钮
